## Công nghệ sử dụng:
### Backend:
* Node.js
* Express.js
* Mongoose
* JSON Web Token
### Frontend:
* Vue.js 3 + Vite
* Tailwind CSS
* Axios
* Vue Router
* Pinia
### Cơ sở dữ liệu
* MongoDB
### Xác thực người dùng
* Clerk
